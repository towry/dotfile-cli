
# dotfile {W.I.P}

A simple cli tool to manage dotfiles.



## Usage

`dotfile -[ars] file`

| option | long     | description                        |
| ------ | -------- | ---------------------------------- |
| -a     | --add    | add file to the dotfiles dir.      |
| -r     | --remove | remove file from the dotfiles dir. |
| -s     | --sync   | sync dotfiles.                     |
| -h     | --help   | show the command help text.        |
